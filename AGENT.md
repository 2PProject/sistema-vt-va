# Agente Desenvolvedor - Sistema VT/VA

Este projeto é um sistema web para gestão de Vale Transporte (VT) e Vale Alimentação (VA)
do Grupo Meire Reis.

Tecnologias do projeto:

- Next.js
- Supabase
- Vercel
- React
- Tailwind

Objetivo do sistema:

Gerenciar empresas, unidades, funcionários e calcular VT/VA mensalmente.

Funcionalidades:

1. Cadastro de empresas
2. Cadastro de unidades
3. Cadastro de funcionários
4. Configuração de competência mensal
5. Cálculo automático de VT e VA
6. Geração de recibos em PDF
7. Relatórios mensais

Regras importantes:

- VT é individual por funcionário
- VA é valor padrão por empresa
- sábados podem ter valor de VT diferente
- feriados geram desconto de VT
- faltas geram desconto proporcional
